# Instrucciones para Instalación y pruebas
## Clonar repositorio e instalación
```sh
$ git clone https://github.com/angelescifuentes/RockettLabs.git
$ cd RockettLabs
$ npm install
```
# Realizacion de pruebas
```sh
$ cd RockettLabs
$ npm test
```

# Inicio servicio
```sh
$ cd RockettLabs
$ npm start
```
Abre http://localhost:3000 y hecha un vistazo
